Web Technologies

HTML

HTML also known as HyperText Markup Language, is a programming Language to create web pages and applications for web pages.
HTML consists of a series of elements and attributes which are used to mark up all the components of a document to
structure it in a meaningful way. It serves as the foundational structure for all content displayed in a web browser.

CSS

CSS is a language we use to design a webpage.
CSS focuses more on style rather than executing logic or programming.
CSS is responsible for adding the color, fonts, and layout to your webpage.

JavaScript

JavaScript is a programming language that allows interactive elements inside of webpages.
It works along HTML and CSS in which it manipulates the content on the page for it to be interactive with the user.
JavaScript enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

AFrame

AFrame refers to an open-source framework for building VR experiences using HTML.
It provides developers with an easy way to create 3D scenes that can run on many different devices such as: a standard desktop, mobile, and VR devices.
By using HTML tags, developers can declare 3D entities like boxes and spheres, making VR content accessible without having to learn complex WebGL code.
