computer programming and web design

The Web Design CTE is a program offered at Bayside Highschool which allows sophmores to explore new concepts using
APIs (Google Maps, Open Weather Map, TheMovieDB, NYC Open Data), libraries (JQuery, mustache.js, c3.js) and data
storage and retrieval (Firebase). Along this, Students will also explore Web Design. Web Design is the process of
planning, conceptualizing, and creating a website's aesthetic look, feel, and functionality to provide a positive user experience.
